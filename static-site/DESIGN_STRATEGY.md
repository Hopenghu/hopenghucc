# 澎湖時光機 - 設計與行銷策略

## 🎨 設計策略

### 品牌視覺識別
```
主色調：澎湖藍 (#0066CC)
輔色調：夕陽橙 (#FF6B6B)
中性色：溫暖灰 (#F8F9FA)
強調色：海洋綠 (#00BFA5)
```

### 設計原則
1. **情感化設計** - 喚起對澎湖的美好回憶
2. **簡潔易用** - 降低使用門檻
3. **社群導向** - 促進用戶互動
4. **回憶聚焦** - 突出分享和重溫

## 🎯 用戶體驗設計

### 核心用戶流程
```
1. 首次訪問 → 快速了解平台價值
2. 註冊登入 → 簡單的社交登入
3. 個人檔案 → 快速設置基本資訊
4. 分享回憶 → 一鍵上傳照片和故事
5. 發現朋友 → 智能推薦相似用戶
6. 計劃重遊 → 與朋友一起規劃
```

### 情感化設計元素
- **溫暖的色調** - 營造親切感
- **圓潤的邊角** - 增加親和力
- **自然的動畫** - 提升愉悅感
- **個人化內容** - 增強歸屬感

## 🎨 視覺設計指南

### 色彩運用
```css
/* 主色調 - 澎湖藍 */
.primary-blue { color: #0066CC; }
.primary-blue-bg { background: #0066CC; }

/* 輔色調 - 夕陽橙 */
.accent-orange { color: #FF6B6B; }
.accent-orange-bg { background: #FF6B6B; }

/* 中性色 - 溫暖灰 */
.neutral-gray { color: #F8F9FA; }
.neutral-gray-bg { background: #F8F9FA; }
```

### 字體選擇
```
主要字體：Noto Sans TC (中文)
輔助字體：Inter (英文)
標題字體：Poppins (強調)
```

### 圖標風格
- 線性圖標為主
- 圓潤的邊角設計
- 一致的視覺重量
- 符合品牌調性
